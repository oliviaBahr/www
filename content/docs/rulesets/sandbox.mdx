## Tags
- Juggler Tag
- Investment Tag

## Spectral cards
- Ouija: old effect changes rank, -1 hand size for rest of game. New effect: Destroys 3 random cards in your hand, then changes cards to a random rank
- Ectoplasm: old effect: 1 negative joker, -1 hand size, hand size reduction increments per use. New effect: 1 negative joker, -1 (hand size / discard / hand). does NOT increment.

## Glass changes
Justice is back. Glass now breaks at 1/3 rate.
